# 自由意志怀疑论的一种语义学回应

张恺璇 2019 年的北京大学哲学系本科毕业论文。

## 文件

- `slides.pdf`：答辩/汇报用演示文稿（中文）
- `thesis.pdf`：本科毕业论文全文（中文翻译 + 英文原文）
- `原创声明.pdf`

## 摘要

本文讨论了自由意志怀疑论，它与外部世界怀疑论的相似性，以及摩尔和维特根斯坦
共同致力的对外部世界怀疑论的解决方案。本文提出了一个与之平行的对自由意志怀疑论
的语义学回应，并辩护了极小自由意志论题（MFT），即某些人在某些时候行使自由意志。
本文论证，任何对 MFT 的怀疑都可以被证明是错的，而任何看起来是对 MFT 的怀疑但不
能被证明是错的怀疑，都是无意义的。


## 引言

我们有自由意志吗？关于这个扣人心弦的问题的争论古老但持久。特别是近几十年来
的科学发展使这个问题再次成为讨论中心（韩水法，2019）。自由意志怀疑论者认为，自由
意志并不存在，因为自由意志与决定论不相容，而决定论很可能是对的。然而，大多数人
觉得自由意志对于我们自己和他人的概念如此重要，以至于几乎不可能抛弃它。
外部世界怀疑论和自由意志怀疑论在论证结构和问题的重要性方面存在诸多相似之
处。在本文中，摩尔和维特根斯坦用于处理前一个问题的反怀疑论策略被应用于后者。本
文辩护了极小自由意志论题（MFT），即某个人在某些时候行使自由意志。文章进一步暗示
极小自由意志论题及其普遍化能够提供值得欲求的那种自由意志。
本文分为三个主要部分。第一部分考虑了“自由意志”一词的两种解释、来自决定论
的威胁以及自由意志的重要性。第二部分讨论了维特根斯坦式无意义，及其如何被应用于
外部世界怀疑论的解决方案。第三部分发展了对极小自由意志论题（MFT）的论证，其使
用了一个思虑（deliberation）的案例和无意义（nonsense）的哲学批评。
